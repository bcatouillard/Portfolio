# Porfolio

Benjamin Catouillard's Portfolio

## Installation

You will need a .env file with : 

```txt
PORT
FORM_ACCOUNT
FORM_PASSWD
FORM_MAIL
API_CAPTCHA_KEY
```

If you want to run this project, you first have to run 3 commands before :

```npm
npm i 
npm run copy:assets
npm run copy:views
```

Then you can run it using : 

```npm
npm start
```
